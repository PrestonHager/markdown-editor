## Unordered List

* Bullet One 
* Bullet Two 
    * Idented Bullet by four or more spaces
* Bullet Three 

Or a list using alternate syntax

- Bullet One
- Bullet Two
     - Indented Bullet
- Bullet Three

## Ordered List

1. Item 1 
2. Two 
3. Three

Or a list using alternate syntax

1) Item 1
2) Item 2 
3) Item 3

## Inline Code

An inline `piece of code` that needs special formatting

## Code Block

An fenced code block

```
import markdown
import json
import base64
```

A code block indented by four or more spaces that is offset:

    import markdown
    import json
    import base64
